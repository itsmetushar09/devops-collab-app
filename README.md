# 🚀 DevTask – DevOps Collaborative Task Management System

## 📌 Project Overview

DevTask is a full-stack web application that simulates a real-world DevOps workflow. It allows users to manage tasks collaboratively while implementing industry-standard practices such as version control, CI/CD pipelines, and secure authentication.

This project demonstrates how modern software teams build, manage, and deploy applications using DevOps methodologies.

---

## 🎯 Objectives

* Build a collaborative task management system
* Implement DevOps practices using GitHub and Jenkins
* Enable CI/CD pipeline automation
* Ensure secure authentication using JWT
* Deploy application to cloud platforms

## 🔄 DevOps Workflow

1. Developer creates an issue
2. Creates a feature branch
3. Implements feature
4. Creates a pull request
5. Code review & merge
6. Jenkins pipeline triggers
7. Application is automatically deployed

## 🧪 How to Run Locally

### 🔹 Clone Repository

```
git clone https://github.com/itsmetushar09/devops-collab-app
cd devops-collab-app
```

### 🔹 Backend Setup

```
cd server
npm install
npm run dev
```

### 🔹 Frontend Setup

```
cd client
npm install
npm start
```
